### UI-04 ~ UI-09: 首次启动向导 (6 步)

> 来源: §2.4 | 状态: ✅ 已实现(5步) / 🔧 扩展中(6步)

**通用框架**:
```
  ┌──────────────────────────────────────────────────────────────┐
  │                    全屏遮罩 (深色背景)                         │
  │                                                              │
  │         ┌────────────────────────────────────────────┐       │
  │         │  ●──●──○──○──○──○  Step 1/6               │       │ ← 步骤指示器
  │         │  ─────────────────────────────────────────  │       │
  │         │                                            │       │
  │         │         ┌────────────────────────┐         │       │
  │         │         │    Step 内容区域        │         │       │ ← 内容区
  │         │         └────────────────────────┘         │       │
  │         │                                            │       │
  │         │  ─────────────────────────────────────────  │       │
  │         │  [ 退出向导 ]      [ < Back ]  [ Next > ]  │       │ ← 底部按钮
  │         └────────────────────────────────────────────┘       │
  │                                                              │
  └──────────────────────────────────────────────────────────────┘
```

**通用样式**:
- 弹窗: 居中, 背景 `#1E2530`, 圆角 12px, 内边距 20px
- 遮罩: 全屏深色覆盖
- 步骤指示器: 6 步圆点 (● 已完成, ○ 未完成) + "Step N/6" 文字
- 底部按钮行: flex row, `LV_FLEX_ALIGN_SPACE_BETWEEN`
  - "退出向导": `btn_secondary`, 全程可见
  - "< Back": `btn_secondary`, 第一步时禁用
  - "Next >": `btn_action (#3B82F6)`, 最后一步变为绿色 "Get Started" `#22C55E`
