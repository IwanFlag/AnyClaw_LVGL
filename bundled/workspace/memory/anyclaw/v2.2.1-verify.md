# PRD 驱动验证清单 — AnyClaw LVGL v2.2.1

> 代码开发已完成。本清单用于逐章对照 PRD + Design 验证功能实现，实测并修 Bug。

## 状态定义

- ⏳ 待验证
- 🔧 验证中
- ✅ 通过（无 Bug）
- 🐛 发现 Bug（附描述）
- 🐛→✅ Bug 已修复

## 工作规则（必须遵守）

1. **每章必须实测**：编译 → Wine 启动 → 截图验证。Wine 没装就装，跑不起来查日志解决
2. **Bug 必须修复→复测→确认** 才算关闭
3. **每章完成后 commit+push**，再进入下一章
4. **同步更新 bundled/workspace/**

## 验证顺序

按 PRD 章节顺序，从 §2 开始（§1 是概览，跳过）。
