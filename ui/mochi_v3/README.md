# 🍡 糯 Mochi v3 — AnyClaw 主题

> 🍡 **主打主题 · 米白系** | 奶茶棕 · 茶道美学 · 文艺质感 | 版本：v2.2.1
> 完整规格见 Design.md §2.4。

## 设计理念

**"温润如玉，静谧如茶，精致如器"**

- 🤎 **米白暖棕**：奶茶棕主色 + 香槟金辅色 + 豆沙粉点缀
- 🫖 **茶道美学**：克制圆角 10-14px + 柔和阴影 + 米白留白
- 📖 **文艺质感**：衬线标题 + 细线图标 + 安静的高级感
- 🎋 **侘寂风**：不张扬、不喧哗，越看越舒服

## 字体

| 类型 | 主选 | 回退 |
|------|------|------|
| 中文 UI | HarmonyOS Sans SC | PingFang SC, sans-serif |
| 中文 标题/引用 | 思源宋体 SC | LXGW WenKai, serif |
| 英文 UI | Plus Jakarta Sans | Inter, Segoe UI, sans-serif |
| 英文 标题/引用 | Lora | Merriweather, Georgia, serif |
| 等宽 | JetBrains Mono | Consolas, monospace |

> 唯一混用衬线体的主题——UI 用无衬线保证可读性，标题和引用用衬线传递阅读质感。
