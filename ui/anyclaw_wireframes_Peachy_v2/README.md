# 🍑 桃气 Peachy — AnyClaw 主题

> 暖色系 · 苹果风 · 萌系 · 年轻人友好

## 设计理念

**"简单但不简陋，活泼但不花哨，温暖但不刺眼"**

- 🧡 **暖色系**：珊瑚橘 `#FF7F50` + 暖橙 `#FFB347` + 粉红 `#FF6B8A`
- 🍎 **苹果风**：大圆角 20-24px + 柔和阴影 + 暖白背景
- 🐱 **萌系元素**：emoji 点缀 + 彩色脉冲动画 + 友好文案
- 💫 **年轻化**：渐变按钮 + 弹性胶囊 + 装饰圆点
