## 1. 产品概述

AnyClaw LVGL Desktop Manager 是一个基于 LVGL 9.6 + SDL2 的桌面应用程序，用于管理和监控 OpenClaw Gateway 服务。

### 1.1 核心价值
- 可视化 OpenClaw Gateway 运行状态
- 一键启停 Gateway 服务
- 中英双语界面支持
- 实时日志显示

### 1.2 技术栈
| 组件 | 版本/技术 |
|------|-----------|
| UI框架 | LVGL 9.6 |
| 窗口系统 | SDL2 |
| 构建系统 | CMake + NMake |
| 编译器 | MSVC 14.50 (VS Build Tools) |
| 字体 | 微软雅黑 (msyh.ttc) + Source Han Sans |
| 操作系统 | Windows 10/11 (x64) |
